# Model Report v2 — Rice Yield + Exogenous Climate (Sierra Leone)

*Target:* **Rice_Yield** (kg/ha) · *Data:* FAOSTAT 2000–2024 + CHIRPS rainfall
+ NASA POWER temperature · *Generated by* `src/run_pipeline_v2.py`

## 1. What changed vs v1

Everything in the v1 design is held identical — the nine complete crops, the
lag-only crop features, the expanding-window walk-forward over
2018–2024, the fixed shallow models, and the
two baselines — so the numbers are directly comparable. The only addition is a
set of **exogenous climate features**, evaluated as three feature sets:
(a) crop-lags only, (b) climate only, (c) crop-lags + climate.

## 2. Climate data

- **CHIRPS v2.0** monthly Africa rainfall GeoTIFFs (2000–2024) were averaged over
  Sierra Leone's geoBoundaries ADM0 polygon (rasterio + the national boundary)
  to a monthly national rainfall series.
- **NASA POWER** monthly `T2M` and `PRECTOTCORR` were sampled at a 5-point grid
  covering the country and averaged into a national monthly series.
- **Cross-check:** CHIRPS vs NASA-POWER monthly precipitation correlate at
  Pearson **r = 0.734** — POWER precip is used only for
  this sanity check, not as a model feature.

## 3. Climate features (same-year permitted — exogenous, observed at harvest)

Growing-season (May–Oct) rainfall total; early-season (May–Jun) rainfall;
peak-season (Jul–Sep) rainfall; growing-season **rainfall anomaly** (z-score vs a
**2000–2017** climatology, computed strictly before the test window); maximum
1-month growing-season **rainfall deficit** vs that month's climatological mean;
mean growing-season **temperature**; and **prior-year** growing-season rainfall
(lag-1). The anomaly/deficit climatology uses only pre-2018 years, so no
look-ahead leakage enters.

## 5. Verdict

**Climate features beat persistence.** The best configuration — **XGBoost on climate** (RMSE 284.1 kg/ha, R² 0.542) — improves on the persistence baseline (RMSE 427.8) by 143.7 kg/ha across the seven held-out years. This reverses the v1 finding, where no model beat persistence.

Best RMSE per feature set — crop-lags **469**, climate **284**, crop+climate **439** kg/ha: the exogenous rainfall/temperature signal carries the gain, and *adding the 35 crop-lag features back in makes it worse* — with ~13–18 training rows they are mostly noise the model overfits.

**Why climate wins — and an honest caveat.** Persistence lags a full year at every turning point (it predicts the 2018 crash value for 2019, the 2022 peak for 2023, …), so it is heavily penalised on this sharply turning series; the climate model instead tracks the yield *level*. The advantage is **not** an artifact of the 2018 outlier — dropping 2018 actually widens the gap (climate 182 vs persistence 438 kg/ha). But note the climate model does **not** actually capture the 2018 crash itself (predicted 1391, actual 786), and rainfall alone is an inconsistent driver across the record — 2020–2022 were below-average-rainfall years yet posted the highest yields (more plausibly Feed Salone input programs than rain).

CHIRPS vs NASA-POWER monthly precipitation cross-check: Pearson r = 0.734, confirming the rainfall series is sound (POWER precip is used only for this check, never as a feature). With only **7 test points** these metrics are high-variance; the result is reported as-is, with no tuning and no feature-chasing.

## 6. Robustness checks (climate-only configuration)

A plain linear model (**Ridge**) on the same climate features and folds scores R² 0.222, RMSE 370.2, MAE 304.0 kg/ha, versus XGBoost's R² 0.542, RMSE 284.1. XGBoost leads Ridge by 86 kg/ha RMSE, so part of the signal is linear but the trees capture additional non-linearity. Importantly, **even the linear model beats persistence** (370 vs 428 kg/ha), so the climate signal is real and not a quirk of one model class. Per-year actual/predicted/absolute-error for every model is in `outputs/per_year_errors.csv`.
